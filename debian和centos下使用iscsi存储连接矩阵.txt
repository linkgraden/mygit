debian挂载多路径iscsi盘阵
1.安装open-iscsi和multipath-tools
apt-get install open-iscsi
apt-get install multipath-tools
-----------------------------------------
2.配置文件etc/iscsi/iscsid.conf
vi etc/iscsi/iscsid.conf
node.startup=automatic
-----------------------------------------
3.重启open-iscsi服务
/etc/init.d/open-iscsi restart
-----------------------------------------
4.搜索iscsi服务器提供的挂载点
iscsiadm -m discovery -t sendtargets -p ip:端口
------------------------------------------------
5.手动挂载
iscsiadm -m node -T iscsi挂载点名称 -p ip:端口 -l
--------------------------------------------------
6.查看挂载情况
fdisk -l
---------------------------------------------------
7.大于2T硬盘修改使用part命令进行修改硬盘格式(小于3T硬盘的直接进行第8步)
parted /dev/挂载盘
mklabel gpt
mkpartfs primary ext4 0 40000 1 -1
----------------------------------------------------
8.初始化挂载硬盘
mkfs -t ext4 /dev/挂载硬盘
----------------------------------------------------
9.挂载硬盘挂载到新创建目录上
mount /dev/挂载硬盘 /新创建目录
------------------------------------------------------
10.查看硬盘挂载到目录情况
df -h
--------------------------------------------------------
11.设置自动挂载方法
vi /etc/fastab
添加 /dev/挂载硬盘 /新加目录 default 0 0
---------------------------------------------------------

注：centos7从第三步骤开始即可。
